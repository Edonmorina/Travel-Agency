# Quote Me!

An application that brings you your needed motivation for the day with hundreds of different quotes by important people from all walks of life!

## Project Status

...This project is currently in development... 

Users can search for different quotes and use keywords to search for specific quotes that they have in mind!

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App Locally:

`localhost:3000/`

To Visit App Remotely:

`quotes-app-gold.vercel.app`
